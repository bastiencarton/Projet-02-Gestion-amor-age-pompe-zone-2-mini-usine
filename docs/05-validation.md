# Validation

## Objectifs des essais

Les essais ont permis de vérifier :

* Le bon fonctionnement de l'amorçage.
* La fiabilité du système.
* L'absence de fuite.
* La répétabilité du fonctionnement.

## Procédure de test

### Test 1 : Amorçage

Objectif :

Vérifier que la pompe principale reçoit correctement l'eau avant son démarrage.

Résultat :

Le circuit est rempli correctement.

### Test 2 : Fonctionnement continu

Objectif :

Valider le comportement du système sur plusieurs cycles.

Résultat :

Le système fonctionne sans interruption.

### Test 3 : Recherche de fuites

Objectif :

Contrôler l'étanchéité globale.

Résultat :

Aucune fuite significative observée.

## Bilan

Les essais montrent que le système répond globalement au cahier des charges défini au début du projet.
